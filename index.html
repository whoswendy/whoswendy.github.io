About Me

Hi, my name is Wendy Hu! I am currently a senior studying Computer Science at Stony Brook University. I live in New York City.

My passion for Computer Science stemmed from my interest in solving problems through the development of applications as well as constantly learning new things within the field. I have always been amazed at the emerging new technology and applications. I remember learning about Microsoft Office and I had so many questions: how was the application developed, where do you even begin, and so forth. This curiosity led me to pursue a career in Computer Science to learn more about the back-end development that goes into applications today, and to develop new technology.

Projects

Outside of academics, I enjoy learning new tech and programming languages through creating small projects and participating in hackathons. I find it most helpful to learn as I go when working on a project. Below are some projects that I have worked on.


SBUHacks2019 - TapNGo
TapNGo was a project that I worked on during the SBUHacks2019 Hackathon.
Target Audience: people with memory disorders or those who just have bad memory.
Functionality: TapNGo is a GPS based app that allows the user to create a list of items that the user or the user's family can set up beforehand to remind them to check if they have the items as they leave the set location. The app notifies the user when they walk out of the set location (i.e., home, supermarket, workplace, etc.).
Technologies Used: Android Studio, Google's FusedLocationProviderClient API, various Android libraries


WordReferenceScrape
WordReferenceScrape was a side project that I worked on alongside my project for the IBM Intern Hackathon.
Target Audience: people who wish to extract french-english conjugation table for other uses
Functionality: An API endpoint that allows one to make a request to WordReference.com and extract the conjugation table for any french-english words.
Technologies Used: Python, Flask, various Python modules such as requests and bs4
